# GTIXT Ethics & Neutrality Charter

GTIXT commits to:

1. **Neutrality**
   - No paid rankings
   - No sponsored scores
   - No preferential treatment

2. **Transparency**
   - Public methodologies
   - Versioned changes
   - Traceable outputs

3. **Independence**
   - No capital allocation
   - No trading activity
   - No commercial influence on scores

4. **Fairness**
   - NA = neutral
   - No penalty for missing non-material data
   - Consistent treatment across regions

5. **Accountability**
   - Public audit trail
   - Error correction via versioning only
   - Clear disclaimers

GTIXT is not competing with prop firms, traders, or platforms. GTIXT defines the rules of measurement.

Whoever defines measurement defines the market.
